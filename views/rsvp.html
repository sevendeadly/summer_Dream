To enable RSVP submissions to Notion:

  1. Create a Notion database with columns: Name, Email, Phone, Attending, Guests, Dietary, Message
  2. Get your Notion API key from notion.so/my-integrations
  3. Share your database with the integration
  4. Add your API key and database ID to script.js

Note: The form will show a message even without Notion setup, but won't save data.
